API ნიშნავს აპლიკაციის პროგრამირების ინტერფეისს. ვებ API არის აპლიკაციის პროგრამირების ინტერფეისი ვების გამოყენებისთვის. ბრაუზერის API-ს შეუძლია გააფართოვოს ვებ ბრაუზერის ფუნქციონირება. სერვერის API-ს შეუძლია გააფართოვოს ვებ სერვერის ფუნქციონირება. ყველა ბრაუზერს აქვს ჩაშენებული ვებ API-ების ნაკრები, რომელიც მხარს უჭერს რთულ ოპერაციებს და ეხმარება მონაცემებზე წვდომას. მაგალითად, გეოლოკაციის API-ს შეუძლია დააბრუნოს ბრაუზერის მდებარეობის კოორდინატები.
