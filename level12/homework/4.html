GitHub

GitHub — Git რეპოზიტორიუმის ვებ-დაფუძნებული კვანძის სერვისი, რომელიც ვერსიის კონტროლის და პროგრამის კოდის კონტროლის საშუალებას იძლევა. GitHub-ს აქვს დამატებული შესაძლებლობები როგორიცაა Wiki, დავალებების მართვა, შეცდომების მიდევნება, და შესაძლებლობების მოთხოვნა ყოველ პროექტზე[2]. GitHub-ი მომხმარებლებს სთავაზობს გადახდილ გეგმებს პრივატული რეპოზიტორიუმებისთვის და უფასო ანგარიშებს, რომელიც ჩვეულებრივ გამოიყენება გახსნილ წყაროიან პროგრამული უწრუნველყოფის პროექტებს. ასევე შესაძლებელია ორგანიზაციების შექმნა რომელშიც მომხმარებლებს შეუძლიათ გაწევრიანდნენ. 2014 წლის სტატისტიკით, GitHub-ს ჰყავს 3.4 მილიონი მომხმარებელი[3], რაც ხდის მას ყველაზე დიდ კოდის კვანძის სერვისს[4].
